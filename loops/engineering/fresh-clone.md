# Fresh clone

> Prove a new dev can go from clone to running app using only the README.

## Goal (the what)

- **Objective:** Verify the setup docs work from scratch.
- **Success criteria:** A clean checkout reaches a running app using only the README.
- **Constraints:** No steps outside the README. Start from a stated baseline (e.g. just the language runtime).
- **Context:** The repo, the README, and the definition of a clean environment.

## Loop (the how)

- **Think:** Read the next README step, decide what it tells you to do.
- **Act:** Run the step exactly as written.
- **Observe:** See whether it works or fails.
- **Reflect / Plan:** When something fails or is missing, fix the README, then start over from a clean clone.
- **Stop when:** A clean environment reaches a running app using only the README.

## Starter prompt

```
Verify our setup docs. Start from a clean clone with nothing pre-installed beyond
[baseline]. Follow the README exactly, step by step. The moment something fails or
needs a step that is not written down, fix the README, then start over from a clean
clone. Stop only when a clean environment reaches a running app using only the README,
with no undocumented steps.
```
